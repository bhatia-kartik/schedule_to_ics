This is a project that takes PNGs of UCLA schedules and outputs an editable table on a web UI of the classes, their days, times and locations. You can then download it as an .ics with a customisable alert. 

Made because I found it boring to manually add my classes to my calendar at the start of each quarter. I've since discovered you can directly download your schedule from myUCLA, but it was still a fun project. 

I used Tesseract and openCV for the detecting part, and vanillaJS for the web UI. I tried deploying but the CV online is too slow to make it practical, so it's best run locally.
